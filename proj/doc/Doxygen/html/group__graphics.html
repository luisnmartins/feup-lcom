void* vg_init ( unsigned short  mode)

Initializes the video module in graphics mode.

Uses the VBE INT 0x10 interface to set the desired graphics mode, maps VRAM to the process' address space and initializes global variables with the resolution of the screen, and the number of colors

Parameters
mode16-bit VBE mode to set
Returns
Virtual address VRAM was mapped to. NULL, upon failure.
